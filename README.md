Shell with minimal features that work (at least for me) and very customizable.

## Features

- Launcher
- Dashboard
- AI Chat (only ollama)
- Bar
- Notifications

and every of them can be disabled.

also you can import config from another person.
or make your own config and widget.

## Requirements

- Hyprland
- imagemagick
- hypridle
- socat
- matugen
- cava
